1. 我們收集的資料

為了提供穩定的機器人服務，糖兒機器人會收集以下運作所需的必要公開與互動資料：

  • Discord 識別碼： 包括您的使用者 ID（User ID）、伺服器 ID（Guild ID）以及頻道 ID（Channel ID）。
  • 指令使用紀錄： 您與本機器人互動時所輸入的斜線指令（Slash Commands）及相關參數內容。
  • 基本公開資訊： 您的 Discord 暱稱、頭像，以及您所在的伺服器名稱與成員列表（僅用於權限驗證與功能呈現）。

1. 我们收集的数据

为了提供稳定的机器人服务，糖儿机器人会收集以下运作所需的必要公开与互动数据：

  • Discord 识别码： 包括您的用户 ID（User ID）、服务器 ID（Guild ID）以及频道 ID（Channel ID）。
  • 指令使用记录： 您与本机器人互动时所输入的斜线指令（Slash Commands）及相关参数内容。
  • 基本公开信息： 您的 Discord 昵称、头像，以及您所在的服务器名称与成员列表（仅用于权限验证与功能呈现）。

1. Data We Collect

In order to provide stable bot services, Tang-er Bot collects the following essential public and interaction data required for operation:

  • Discord Identifiers: Including your User ID, Server ID (Guild ID), and Channel ID.
  • Command Usage Logs: The Slash Commands and related parameters you input when interacting with the Bot.
  • Basic Public Information: Your Discord username, avatar, and the names/member lists of servers you share with the Bot (used only for permission verification and feature rendering).

2. 資料的使用方式

我們所收集的資料僅會用於以下合法且合理的用途：

  • 維持本機器人各項核心功能的正常運作與邏輯判斷。
  • 追蹤與修復機器人運作時產生的程式錯誤（Bugs）與崩潰問題。
  • 防範與阻止惡意使用者利用本機器人進行系統干擾或違反 Discord 社群守則之行為。

2. 数据的使用方式

我们所收集的数据仅会用于以下合法且合理的用途：

  • 追踪与修复机器人运作时产生的程序错误（Bugs）与崩溃问题。
  • 防范与阻止恶意用户利用本机器人进行系统干扰或违反 Discord 社群守则之行为。

The data we collect is used solely for the following legitimate and reasonable purposes:

  • Preventing and stopping malicious users from disrupting the Bot's system or violating Discord Community Guidelines.
